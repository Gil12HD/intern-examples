dojo-example
=============

It is based on the [TodoMVC Dojo Example](http://todomvc.com/examples/dojo/).

## Setup

1. Install the JRE or JDK. This demo uses Selenium, which requires Java, to run WebDriver tests.

2. Install node modules
   ```
   $ npm install
   ```

## Running tests

**Unit tests in Node**

    $ npm test

**WebDriver tests**

    $ npm test webdriver
